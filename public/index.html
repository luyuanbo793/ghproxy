GitHub Proxy

🎊现已支持 GitHub API (api.github.com) 代理🎊

支持终端命令行 git clone , wget , curl 下载

GitHub文件链接带不带协议头都可以，支持 releases、archive、gist、raw.githubusercontent.com 文件（除文件夹）
