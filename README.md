# ghproxy

这是 [gh-proxy](https://github.com/hunshcn/gh-proxy) 的 Go 语言版本。

## 支持内容

• GitHub API (api.github.com)

• Raw 文件 (raw.githubusercontent.com)

• Gist 文件 (gist.githubusercontent.com)

• Releases 文件 (github.com/用户名/仓库名/releases|archive/)

• Git 克隆支持 (git clone)
